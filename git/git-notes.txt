## 开始使用git
1. CMD open the directory of C:\Program Files\Git\usr\bin
2. Key in:     ssh-keygen -t rsa -b 4096 -C "xxx@xxx.com"
3. C:\Users\username\.ssh\id_rsa.pub 的内容在 github Account SSH
4. 测试: ssh -T git@github.com -> Hi XXX! You've successfully authenticated, but GitHub does not provide shell access.
5. git config --global user.name "your name"
6. git config --global user.email "your_email@youremail.com"

7. 去到要上传的folder下， git init -> 生成隐藏的.git folder
8. git remote add origin git@github.com:FHeaven/study-notes.git 关联到github
